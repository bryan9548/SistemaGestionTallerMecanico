# Sistema de Gestión de Taller Mecánico

## Descripción
Proyecto Final del curso de Programación Orientada a Objetos desarrollado en Java con NetBeans y SQL Server.

## Tecnologías utilizadas
- Java 24
- NetBeans 25
- SQL Server
- JDBC
- Maven

## Funcionalidades
- Gestión de clientes
- Gestión de vehículos
- Gestión de mecánicos
- Gestión de repuestos
- Gestión de órdenes de trabajo
- Reportes
- Gestión de empleados
